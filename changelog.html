v2.0.0

    NOTE If you're coming from an earlier version of this add-on:

  • Added feature to show ordered images
    • Only works on show answer pop-up
    • Image counter is global (not deck based) and won't reset until you close Anki
  • Added native image resizing
    • Now the maximum height and width for large images is 1024px
    • Larger images will shrink to this size
    • The aspect ratio of the images will be preserved
  • Separated Audio folder from Video/GIF folder
  • Now you can't have both header/image pop-up and video/gif pop-up enabled at the same time
  • Removed pop-up button and related items in the settings
  • Removed pop-up window title and related items in the settings
